同源策略限制

跨域指的是当两个url的协议，域名，端口三者有一个不同，就会发生跨域。之所以会有跨域，是因为浏览器的同源策略限制。
同源策略限制了一个源的脚本与另一个不同源的脚本之间的交互，这是一种安全保障机制。
是否同源的判断，只要看两个源的url端口和域名是否相同，只要有一个不同，那就是不同源。
