职责链模式的定义

职责链模式将多个对象用链的形式连接起来，当某个对象无法执行某个方法时，将请求沿链发送到下一个对象，交给下一个对象去处理，直到对象被处理或者到达链的末端，而链的末端一般是表示方法无法被执行或者标明某种特殊情况。
